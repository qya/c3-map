Privacy Policy

Last updated: May 20, 2026

What C3 Code Processes

C3 Code converts map clicks, pasted coordinates, C3 codes, and H3 indexes into geospatial cell data in your browser. Location search input is used only to update the current map view.

Location Permission

If you use the locate button, your browser may ask for location permission. The resulting coordinate is used to move the map and calculate the current C3/H3 cell.

Third-Party Map Services

Map tiles and reverse geocoding may be requested from third-party providers. Those providers can receive request metadata such as IP address, approximate map area, and user agent.

Local Storage

The app stores your light or dark theme preference in browser local storage. It does not require an account.

Contact

For privacy questions, update this page with the production contact address before launch.
